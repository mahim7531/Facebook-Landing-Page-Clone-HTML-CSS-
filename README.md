# 📘 Facebook Landing Page Clone

A simple Facebook landing page clone built using **HTML & CSS**.

## 🚀 Features

* Clean Facebook-style UI
* Responsive Flexbox layout
* Login form design
* Hover effects
* Modern and professional look

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox)

## 📁 Project Structure

```
facebook-landing/
│── index.html
└── style.css
```

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser

## 📌 Purpose

This project is created for practice and learning front-end development.
